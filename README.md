# BA-RAW-TEXT

A places for untranslated text content used as a reference.
This branch data has been compiled and filtered to only show content that has not yet been translated in Global as part of the [BA-TL-TEXT](https://github.com/Special-Operation-Decagrammaton/BA-TL-TEXT) project for translation.

If you want to translate and contribute, please see [BA-TL-TEXT](https://github.com/Special-Operation-Decagrammaton/BA-TL-TEXT) and read the [wiki](https://github.com/Special-Operation-Decagrammaton/BA-TL-TEXT/wiki) for more information.
